![image](https://user-images.githubusercontent.com/81466838/142440343-0a94bafa-e0bf-466f-8e74-3a4203384ce3.png)
## CTRL + P to access Nuget Package Manager GUI
![image](https://user-images.githubusercontent.com/81466838/142440631-073506b8-22b1-4192-aa64-a2f196d7d77e.png)
## On the Nuget Package Manager GUI, go to install new package, search for Microsoft.AspNetCore.Mvc.NewtonsoftJson and install version 3.1.20
![image](https://user-images.githubusercontent.com/81466838/142441204-6a7d63df-e062-4a9b-b6e3-99d861224fd8.png)
## Then Search for npgsql and install the latest version
![image](https://user-images.githubusercontent.com/81466838/142441524-815bac13-dc80-47d8-9b28-f44336e5943f.png)
## Dotnet run on terminal to start the program.
## After setting up, you need to go to localhost:5001(Or whatever address your backend is allocated to) to allow the certificate as the backend api is in HTTP.
